# docs ディレクトリについて

このディレクトリには、米国株式市場の要約レポートおよび投資アイデアをまとめたMarkdownファイルが日付ごとに格納されています。

## ファイル命名規則

- `market_summary_YYYYMMDD.md` 形式（例: `market_summary_20250808.md`）
- 各ファイルには、その日の米国株市況の要約と投資アイデアが記載されています。
